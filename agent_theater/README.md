# Agent Theater

Agent Theater displays safe, human-readable summaries only. It must not reveal hidden chain-of-thought, secrets, broker credentials, tokens, passwords, or raw paid-LLM prompts.

Implemented rooms:

- Live Chat View
- Workflow Timeline
- Debate Mode
- Boardroom Mode
- Strategy War Room
- Account Routing Room
- System Improvement Room

The API stores canonical English events and can render user-facing labels in English or Bahasa Melayu Malaysia. Trading symbols, code terms, JSON keys, environment variables, and IDs remain canonical.
